# NamRent

NamRent is a React and Node.js rental platform prototype for Windhoek, Namibia. It includes searchable rental listings, student accommodation, short stays, adverts, safety guidance, and a lightweight Express API scaffold.

## NamRent Front End

- React with Vite
- Homepage hero search with styled dropdowns
- Rental flashcard carousel
- Featured, student, family, and short-stay rental views
- Property detail pages
- Advert carousel
- Chatbot assistant UI
- Mobile-friendly layout

Run locally:

```bash
npm install
npm run dev
```

## NamRent Back End

- Node.js with Express
- Sample listing, location, advert, and enquiry endpoints
- Ready to connect to a real database later

Run the API:

```bash
npm run server
```

Default URLs:

- Front end: `http://127.0.0.1:5173`
- Back end: `http://127.0.0.1:4200`

## Build

```bash
npm run build
```
